## Get Started



### 1.Clone & Install

```shell
git clone git@github.com:unira-zwj/LightURDI.git
```

你需要预先安装ROS2

---


### 2.Start the pos pub node

```shell
bash start_env.sh
```


### 3.Start robot arm tele opration

```shell
python ur3_follow_with_web_ui
```
